Web Resources

  1. Google Web Fundamentals
    Web Fundamentals is a Google project to provide free, open source, online coding lessons and tutorials to the masses. This website has great article on respnsive web design, as well as other web development related articles.

    Rating: 4

  2. Sololearn
    SoloLearn is an online and mobile learning platform with free coding classes in 13 different programming disciplines It also has courses on CSS, Responsive Web Design, SQL, React, and many others if web applications aren't the only thing you want to learn. As a full stack developer, this website offers a plethora of courses in many different programming languages which you might require.

    Rating: 4

  3. Hacker.io
    Hackr.io is a site that compiles lists of the favorite programming courses and tutorials from across the web. You can also use their search engine and upvoting system to find courses on any type of delopment and programming language you want. If you don't know where to find courses to learn programming, this website's search engine helps you search for all the courses that are offered online on whatever specific language you desire.

    Rating: 4.5

  4. Nodeschool.io
    Nodeschool.io is a platform with open source workshops on Node.js that you can do online all on your own. You can start with the basics of learning JavaScript and try their core workshops on Nodejs. You can download the workshops from github and run them on your own computer as an executable file. It is a very interesting way to learn coding wihtout having to do it through a web browser like most other coding courses.

    Rating: 3.75

  5. Exercism.org
    Practice problems for new coders and experts, tailored for your interests, whether you want to learn to code, increase fluency in your chosen language, or develop clean code. You can submit your solutions and get helpful feedback. It also works through GitHub. Very handy website where you can upload your work and get real time help. Similar to stackoverflow.

    Rating: 4
